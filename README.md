## 📱 Vue d'ensemble

Application mobile complète pour surveiller et gérer les zones à risque d'inondation en temps réel.

## 🎯 Fonctionnalités

- 🗺️ Visualisation des zones inondables
- 🚨 Système d'alertes temps réel
- 👤 Authentification sécurisée (JWT)
- ❤️ Favoris et suivi des zones
- 📊 Historique des niveaux d'eau
- 📍 Géolocalisation GPS

## 🚀 Démarrage Rapide

```bash
# 1. Backend
cd floodmap-backend && npm install && npm run dev

# 2. Émulateur
emulator -avd Pixel_5

# 3. App
flutter pub get && flutter run
```
